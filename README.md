# Gorisanson Repo

## How do I install these formulae?

`brew install gorisanson/repo/<formula>`

Or `brew tap gorisanson/repo` and then `brew install <formula>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
